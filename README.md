# PPTGenerator

A Python script that automatically generates a PowerPoint presentation from a video file by extracting key frames.

## Usage

To use the script, run `main.py` from the command line and provide the path to a `.mp4` video file as an argument.

```bash
python main.py "path/to/your/video.mp4"
```

For example:

```bash
python main.py "C:\Mahesh\AI projects\pptgen\uploads\S2S.mp4"
```

## Output

The generated PowerPoint presentation (`.pptx` file) will be saved in the `generated` folder.
